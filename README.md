# locker
Sperre deinen Windows-Rechner, während du nicht da bist!<br/><br/>

Dieses Programm wird bei jedem Start ausgeführt,<br/>
sobald du es installiert hast (Installer mitgeliefert)<br/>
,protokolliert jeden Bootvorgang<br/>
und fährt den Computer sogar herunter, wenn du es so eingestellt hast und<br/>
nicht innerhalb von 10 Sekunden das zuvor gewählte Passwort eingegeben wird.
